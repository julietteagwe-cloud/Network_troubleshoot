# Network Troubleshooting Script
This repository contains my network troubleshooting script used for Ubuntu and Centos
# Author : Juliette Joseph-Odogbo
# Purpose
The script checks
- IP address configuration
- Default gateway settings
- DNS resolver settings
- Internet connectivity (ping test)
It also generates a result stored in the Report text file
# How to Run the Script
1. Make the script excutable
   chmod +x network_troubleshoot.sh
2. Run it with sudo
   sudo ./network_troubleshoot.sh

# Output
network_troubleshoot_report.txt
